# Dataset Information

## Dataset Title: Full TMDB Movies Dataset 2024

### Description:
This dataset contains information about movies from the TMDB (The Movie Database) spanning various attributes such as titles, genres, ratings, release dates, revenue, and more. It is useful for conducting analyses in movie ratings, box office performance, genre distribution, and other related areas.

### Size:
- Total Movies: Approximately 1,000,000 (1M)

### Columns Included:
- `id`: Unique identifier for each movie
- `title`: Title of the movie
- `original_language`: Language in which the movie was originally released
- `release_date`: Release date of the movie
- `vote_average`: Average rating of the movie
- `genres`: Genre(s) associated with the movie
- `revenue`: Revenue generated by the movie
- And many more...

### Data Source:
- [TMDB Movies Dataset 2023](https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies)

### Usage:
You can access the dataset from Kaggle using the link above. Please note that you need a Kaggle account to download the dataset.
